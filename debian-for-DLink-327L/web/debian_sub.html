Debian for DLink

Debian for DLINK was done by 8ohmh and vterdohleb@dlink

Debian for DLink contains an almost full Debian distro for this DLink NAS, full upgrade- and update-able. Missing due Hardware is the X11 Server/Graphics output but can be by implemented by buying an USB Graphics Card/adapter

How to use:

1. Make sure that SSH is enabled by Admin Page -> Management -> Network Management -> SSH

2. For Windows, download OpenSSH and Putty, and install it, then read how to use it to connect to this NAS

3. For Linux, just install ssh by apt-get install sshd and login into this NAS

4. The Debian Linux starts automatically and does so far not intefere much with the original NAS Linux.

5. On command line do sh /chroot2debian.sh

6. To use it at best, install by apt-get install mc (Midnite Commander),screen and the same addons provided by DLINK
